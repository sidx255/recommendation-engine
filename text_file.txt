The pattern that is searched in the file is referred to as the regular expression (grep stands for globally search for regular expression and print out).<strong>&nbsp;</strong><br><strong>Syntax:</strong>&nbsp;<br>&nbsp;<pre><strong>grep [options] pattern [files]</strong></pre><p>&nbsp;<pre><strong>Options Description</strong>
<strong>-c</strong> : This prints only a count of the lines that match a pattern
<strong>-h :</strong> Display the matched lines, but do not display the filenames.
<strong>-i :</strong> Ignores, case for matching
<strong>-l :</strong> Displays list of a filenames only.
<strong>-n :</strong> Display the matched lines and their line numbers.
<strong>-v :</strong> This prints out all the lines that do not matches the pattern
<strong>-e exp :</strong> Specifies expression with this option. Can use multiple times.
<strong>-f file :</strong> Takes patterns from file, one per line.
<strong>-E :</strong> Treats pattern as an extended regular expression (ERE)
<strong>-w :</strong> Match whole word
<strong>-o :</strong> Print only the matched parts of a matching line,
 with each such part on a separate output line.

<strong>-A n</strong> <strong>:</strong> Prints searched line and nlines after the result.
<strong>-B n :</strong> Prints searched line and n line before the result.
<strong>-C n :</strong> Prints searched line and n lines after before the result.</pre><p>&nbsp;<p><strong>Sample Commands</strong><p>Consider the below file as an input.&nbsp;<br>&nbsp;</p><br><div id=AP_G4GR_5></div><div id=d6f45c53-ff16-4cc5-8b91-eb6ee7037116 class=_ap_apex_ad><script>var adpushup = window.adpushup  = window.adpushup || {que:[]};adpushup.que.push(function() {adpushup.triggerAd("d6f45c53-ff16-4cc5-8b91-eb6ee7037116");});</script></div><br><pre><strong>$cat &gt; geekfile.txt</strong></pre><p>&nbsp;<pre>unix is great os. unix is opensource. unix is free os.
